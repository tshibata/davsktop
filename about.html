About Davsktop

Davsktop = WebDAV + Desktop

Just put some files on your WebDAV server. It will make your browser a WebDAV client with desktop LnF.

Tips and Tricks

  • You can rename a file by drag-and-dropping it into the same folder. Davsktop asks you new name whenever the destination folder already has a file with the same name.
  • Link icon() is just a hyper link to file. It allows you anything you can do with your browser, e.g. download, copy URL, open in new tab.

Known Problems

  • Can't copy on Chrome.
  • Can't upload edited text on Chrome.
  • Can't drag-and-drop files to local desktop on Firefox.
  • Too many problems on IE9 to list. Not tested on IE10.
