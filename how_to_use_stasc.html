Generate HTML Element

Let's suppose you need to generate a table for translation on a web page.
For example, "State" is "Staat" in German. So your table will have a row like following: 
<tr><td>State</td><td>Staat</td></tr>
This means you need to generate an element which has two internal elements with text nodes. In javascript it is done by following: 
var row=document.createElement('tr');
var n1=document.createElement('td');
n1.appendChild(document.createTextNode('State'));
row.appendChild(n1);
var n2=document.createElement('td');
n2.appendChild(document.createTextNode('Staat'));
row.appendChild(n2);
Anybody can understand your intention from the code, but it is not intuitive compared to the original tags.
Stasc translates such html tags into javascript functions.
Save the tags as StateInGerman.ss.xml and run stasc. Then you will have "Stasc.js" which contains function named after the source file. You can generate the element like: 
var row=StateInGerman();

Generate More Than One HTML Element

You can give stasc more than one source file to let "Stasc.js" have more than one function. The source files can be given as parameters, for example: 
stasc StateInGerman.ss.xml ScribeInGerman.ss.xml
If no source file is given, it picks "*.ss.xml" in current directory.

Refer HTML Element

You have to put the row into your table. Of course you can use stasc also to generate the table. 
<table>
<thead>
<tr><td>English</td><td>Deutsch</td></tr>
</thead>
<tbody>
<!-- Body -->
</tbody>
</table>
In the table, tbody has a comment. This allows you to refer the tbody as Body. So you can put your row in the tbody. 
var table = EnglishGerman();
table.Body.appendChild(StateInGerman());

Refer HTML Attribute

Your table must have more words. 
<tr>
<td><span><!-- Left=textContent --></span></td>
<td><span><!-- Right=textContent --></span></td>
</tr>
By writing like this, you can refer textContent of the first span as "Left" and the second as "Right". 
var table=EnglishGerman();
var row;
row=LeftRight();
row.Left="state";
row.Right="staat";
table.Body.appendChild(row);
row=LeftRight();
row.Left="scribe";
row.Right="schreiber";
table.Body.appendChild(row);

Replace HTML Element

In some cases, you may want to replace element. 
<table>
<thead>
<tr>
<!-- Head ! -->
</tr>
</thead>
<tbody>
<!-- Body -->
</tbody>
</table>
Note that "Head" is followed by an exclamation mark. That means you can not only refer but also change the element. Now you can use the table for any purpose. 
var table=Table();
var row;
row=LeftRight();
row.Left="English";
row.Right="Japanese";
table.Head=row;
row=LeftRight();
row.Left="state";
row.Right="kuni";
table.Body.appendChild(row);
